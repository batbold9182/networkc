Fullstack currency convertor built with ReactNative/expo , Node.js/Express for the backend with MongoDb
#########################################################################3
Featuring real time exchange rates from the National Bank of Poland (NBP)
#########################################################################3
Features :
User Authentication - Secure login/signup with JWT tokens
Digital Wallet - Manage multiple currency balances
Currency Exchange - Buy and sell currencies at real-time rates
Historical Rates - View historical exchange rate charts
Account Funding - Add PLN balance to your account
ransaction History - Track all your exchange transactions
############################################
Tech Stack
FrontEnd ReactNative Expo        ReactNavigation Axios 
BackEnd  Node.js     express.js  JWT auth        NBP API   
DataBase MongoDb     Mongoose
